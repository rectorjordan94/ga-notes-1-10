## Mini Algorithm Exercise

"Does Include"

### Objectives

- Check if a given parameter (`x`) exists in a given array or string (`data`) **without using a for or while loop**.
- The output should either return `true` or `false` depending on the test case.

### Getting Started

- Fork and clone this repo.
- Run `npm i` from within the repo.
- Run `npm run test` or `npm test` to make sure the tests work. (They should fail.)
- Get hacking on the problem, test when you think you've got it, and use your tools to debug and find solutions.

### Notes

The objective of this problem is to assess your creativity and resourcefulness in solving a restrictive problem.
